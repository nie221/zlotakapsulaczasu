
# Złota Kapsuła Czasu V6

Nowa wersja z automatycznym nasłuchiwaniem zdarzeń na zakładki!

## Funkcje
- 🕰 Historia wspomnień z ukrytą zawartością przed odblokowaniem
- 📝 Dodaj nowe wspomnienie (z własną ikoną i dowolną datą odblokowania)
- ℹ️ O aplikacji
- Piękny ciemny motyw z zielonymi akcentami
- Automatyczna obsługa kliknięć w menu

## Jak uruchomić?
1. Otwórz `index.html` w przeglądarce.
2. Klikaj w zakładki w menu bocznym.
